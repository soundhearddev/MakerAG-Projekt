Wie Fügt man Weitere Gegenstände hinzu?

Um weitere Gegenstände hinzuzufügen muss eine Verbindung mit dem Server hergestellt werden.

Um dies zu tun, vergewissern Sie sich, dass Sie die benötigten SSH Keys besitzen.

Zum Verbinden öffnen Sie das Terminal und führen Sie diesen Befehl aus:

ssh -i /path/to/your/key.pem -L 80:127.0.0.1:80 -N maintain@10.1.200.8 -p 222

Wenn dies getan ist, können Sie Ihren Browser öffnen und auf die Seite http://localhost navigieren.
